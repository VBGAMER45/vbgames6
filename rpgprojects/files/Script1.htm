Project X Scritping

 

Introduction: In this guide you learn how to add custom events to Project X and learn all the scripting functions!

You script in vbscript using the following objects. Red is an Object. Blue is the code. Green are comments

The Main form - This is the main form where all the game action happens

Objects

MainForm - all properties of a form

Hero - all properties of a image control. This is the main charecter

MagicBar - all properties of a image control. Shows the status of the magic

Magic - all properties of a label control. The caption property ranges from 0-8. Zero as the lowest

Money - all properties of a label control. The caption property holds how much money you have!

ChatBox - This is a custom control! To set the Chat Text just set chatbox.messagetext = "YourTextHere"

MapName - all properties of a label control. The caption property holds the mapname

Music - all properties of a mediaplayer control.

Sound - all properties of a mediaplayer control.

Hscroll - all properties of a scrollbar. This scrolls the map

Vscroll - all properties of a scrollbar. This scrolls the map

CheatMenu - all properties of a menuitem.

Direction - all properties of a label control. The caption property holds what direction the charecter is going

MNUSave - all properties of a menuitem. You can use this to disable saving in a certain level or room

ChatBoxTimer - all properties of a timer control. How long the chatbox appears using the interval property

Panel - all properties of a frame control

CurrentWeapon - all properties of a image control

CurrentSpell - all properties of a image control

CurrentItem - all properties of a image control

BoxTop - all properties of a label control. The caption property holds The Y location of the charecter and what square the hero is on!

BoxLeft - all properties of a label control. The caption property holds The X location of the charecter and what square the hero is on!

Hearts - all properties of a label control. The caption property holds how many Hearts the hero has between 0-6

ArrowSpeed all properties of a timer control. How fast the arrow goes using the interval property

SnowTimer all properties of a timer control. How fast the snow moves using the interval property

Examples for the objects on the mainform

How to disable Saving:

Mnusave.enabled = false

How to stop the music and sound effects

Music.Stop

Sound.Stop

How to play the music and sound effects

Music.Play

Sound.Play

How to change the music

music.FileName = "c:\windows\desktop\newfolder\z5oot[2].mid"

A Trigger or Simple Event Example

if mapname.caption = "map1.map" Then

If boxtop = "1" and boxleft ="2" then 'Checks if hero is in this square

chatbox.visible = true

chatbox.messagetext = "You found a treasure chest!!!"

chatboxtimer.enabled = true

money.caption = money.caption + 300

xp.caption = xp.caption + 50 'You will learn the xp object in the next section

end if

end if

How to change a spell

CurrentSpell.tooltiptext = "warp!!" 'This makes the current spell the warp spell

The Item Select & Stats form - This is the form that holds the items,weapons,spells, and stats

Objects

ItemScreen - all properties of a form

ItemSelectTitle- all properties of a label control The caption property holds the Title of the screen

XP - all properties of a label control. The caption property holds how much experience the hero has

XP4NextLvl - all properties of a label control. The caption property holds how much experience till the next level!

Kills - all properties of a label control. The caption property holds how many kills the player has!

Level - all properties of a label control. The caption property holds what level is the hero

SpendXP - all properties of a label control. The caption property holds how xp the player can spend

Examples for the objects on the Item Select and Stats form

Here is a quick example of what you can do!

if level.caption = "20" then

chatbox.visible = true

chatbox.messagetext = "You have reached a level where you will receive an XP Prize"

chatboxtimer.enabled = true

xp.caption = xp.caption + "1000"

spendxp.caption = spendxp.caption + "500"

xp4nextlvl.caption = xp4nextlvl.caption - "1000"

hero.top = "400"

hero.left = "400"

magic.caption = "8"

music.FileName = "c:\windows\desktop\newfolder\prize.mid"

sound.stop

money.caption = money.caption + "300"

end if
