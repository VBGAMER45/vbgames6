The PopUp Boxes Form- This is the makes custom msgbox like form

Objects

PopBox - all properties of a form

PopBoxTitle - all properties of a label control. The caption property holds the title of the box

PopBoxText - all properties of a label control. The caption property holds the text of the box

PopBoxShape1 - all properties of a shape control

PopBoxShape2 - all properties of a shape control

PopBoxButton1 - all properties of a label control. The caption property holds the text of the button

PopBoxButton2 - all properties of a label control. The caption property holds the text of the button

PopBoxClose - all properties of a image control.

PopBoxImage- all properties of a image control.

PopBoxFrame1 - all properties of a frame control.

PopBoxFrame2- all properties of a frame control.

Example PopBox

popbox.show

Popboxtitle.caption = "Test"

popboximage.visible = false

popboxframe1.visible = true

popboxframe2.visible = true

PopBoxtext.visible = true

PopBoxtext.caption = "This is just a test"

Popboxbutton2.visible = false

Popboxshape2.visible = false

Popboxbutton1.caption = "Close"
